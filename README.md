# Napredna-analiza-podataka---Veliki-jezicki-modeli
Ovde je uradjen projekat iz predmeta Napredna analiza podataka. Tema projekta jeste ekstrakcija informacija iz recenzija hotela koriscenjem LLM i dalje koriscenje tih podataka za napredne analize. Uradjena je mrezna analiza recenzija prema dobijenim podacima.
